统一管理 AI 与 PDF 提取服务参数

在这里配置 AI 模型参数、Prompt 模板和 PDF 提取服务，所有设置统一管理。

AI 配置

先维护模型池，再把不同模型绑定到问答、图片解读、标签建议、分组建议等场景。

模型池

点击模型名按钮即可修改；新增模型会先弹出配置窗口，保存后再加入模型池。

场景绑定

为不同 AI 场景选择默认模型；如果没有特别指定，系统会回退到默认模型。

Prompt 使用说明

System Prompt 是全局角色设定，作用于所有 AI 场景。

通用问答 Prompt 面向 AI伴读页的单篇文献问答。

当前图片解读 Prompt 和 图片标签建议 Prompt 都只作用于当前正在查看的单张图片，不是整篇文献的所有图片。

分组建议 Prompt 作用于整篇文献的分组判断。

论文标题、摘要、备注、全文、现有标签/分组、当前图片信息，以及 JSON / 流式输出要求都会由后端自动补充，这里只需要写额外的行为指令。

PDF 提取服务配置

配置 PDF 提取服务的连接信息，用于上传时自动解析文献。

填写 `http://127.0.0.1:8000` 这样的服务根地址即可；系统会自动补全 ` /api/v1/extract `，并始终从同一根地址推导 ` /api/v1/jobs `。

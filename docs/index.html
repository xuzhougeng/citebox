核心功能

为学术研究而生的每一处细节

不只是存储论文——CiteBox 让你与文献建立更深层的对话

智能 PDF 解析

上传论文，自动提取标题、摘要、正文与图表。全文索引让你再也不会遗忘任何一个关键信息。

图表管理与子图标注

自动提取论文中的每一张图表，支持子图标注、图注编辑和独立标签分类。建立你的学术图库。

AI 伴读助手

与 AI 聊论文——提问、翻译、解读图表。支持 OpenAI 等多种模型，让每篇论文都成为一场深度对话。

灵活的组织方式

用分组和标签自由组织文献。彩色标签、嵌套分组，以你最习惯的方式管理每一篇论文。

色彩提取

从论文图表中提取配色方案，保存到你的色板库。科研绘图时直接引用，保持视觉一致性。

桌面原生体验

支持 Windows、macOS、Linux 三大平台。本地部署，数据完全掌控在你自己手中。
